# Guia de construção do README:

# 1. **Introdução:** Breve descrição do projeto e seus objetivos.
# 2. **Funcionalidades:** Lista das principais funcionalidades do plugin.
# 3. **Instalação:** Passo a passo para instalar o plugin no Zotero 8.
# 4. **Uso:** Instruções básicas de como usar o plugin para gerar referências.
# 4.1 **Exemplos:** Exemplos de referências geradas pelo plugin, comparando com as referências esperadas (link para `docs/referencias-ufc-exemplos.md`). Deve listar TODAS as referências, os campos que devem ser preenchidos no Zotero, exemplos de referência e dicas de como encontras as informações necessárias (exemplo: "Caso o Zotero não encontre o ISBN do seu livro, as informações estão no expediente (que costuma ser a página tal)");
# 5. **Contribuição:** Guia para desenvolvedores que queiram contribuir com o projeto (link para `CONTRIBUTING.md`).
# 6. **Licença:** Informação sobre a licença do projeto (ex: MIT, GPL, etc.).
# 7. **Contato:** Informações de contato para dúvidas ou suporte.
